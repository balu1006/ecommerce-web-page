# Ecommerce Web Page

Simple, static e-commerce landing page built with HTML and CSS.

## Overview

This repository contains a lightweight, responsive front-end for an e-commerce website. It's intended as a template or demo to showcase product listings, features, and promotional sections.

## Features

- Responsive layout using CSS
- Hero/banner, product sections, features, and blog/people sections
- All assets (images, CSS) are included in the repository

## Files

- `index.html` — main entry point for the site
- `css/style.css` — stylesheet for the site
- `images/` — image assets organized into subfolders

## How to view

1. Open the project folder in your file browser or editor.
2. Open [index.html](index.html) in a web browser (double-click or use "Open File").

## Development

- Edit HTML in `index.html` and styles in `css/style.css`.
- Replace or add images under the `images/` folder as needed.
